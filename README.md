# 📊 Power BI Report — sample2

## Overview
This repository contains a Power BI Desktop report (`sample2.pbix`) built using Microsoft Power BI.

## 📁 File Structure
```
├── sample2.pbix       # Power BI Desktop report file
└── README.md          # Project documentation
```

## 🚀 Getting Started

### Prerequisites
- [Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free download)

### Opening the Report
1. Clone or download this repository
2. Open **Power BI Desktop**
3. Click **File → Open report**
4. Select `sample2.pbix`

## 📌 Requirements
| Tool | Version |
|------|---------|
| Power BI Desktop | Latest recommended |
| Windows | 10 / 11 |

## 🔗 Publishing to Power BI Service
1. Open the report in Power BI Desktop
2. Click **Home → Publish**
3. Select your Power BI workspace
4. Access the report at [app.powerbi.com](https://app.powerbi.com)

## 📝 Notes
- `.pbix` files are binary — consider using [Git LFS](https://git-lfs.github.com/) for large files
- Add `*.pbix filter=lfs diff=lfs merge=lfs -text` to `.gitattributes` to enable LFS tracking
